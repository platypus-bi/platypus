## Deploy it on your own

### Requirements

To run Platypus, you need the following:

#### Server

- A recent version of Docker (with Docker Compose)
- At least 2 GB of RAM

#### Your PC

- An installation of [Microsoft Power BI Desktop](https://aka.ms/pbidesktopstore)
